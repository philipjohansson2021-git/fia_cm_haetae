# 실험 / 시뮬레이션 로그 (날짜별)

## 구성
- `YYYY-MM-DD/` — 날짜별 결과·리포트·그림.
- `YYYY-MM-DD/sim_code/` — 그날 사용한 시뮬레이션 코드(대응기법별 정리: `by_countermeasure/`
  + 동작 통합본 `common/`, 레퍼런스 소스 동봉 자체완결 빌드).

## 관리 규칙
- 새 실험: `test/<오늘날짜>/` 생성 → 산출물 저장 → 그 폴더 `README.md` 요약 → 이 인덱스에 한 줄 추가.
- 커버리지(논리/실측 공통)는 아키텍처 독립 1회; 절대 오버헤드·공격 성공률은 하드웨어(F3/F4) 실측으로 별도 기록.
